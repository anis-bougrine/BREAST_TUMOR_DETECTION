# BRESAT_TUMOR_DETECTION
In the beginning i shoosed the RESNET50 of Keras as pretrained model for classification and after applying transfer learning i had an overfitting problem that it hasn't been resolved even after applying overfitting techniques such as dropout (97% train accuracy VS 80% test accuracy), so i switched to a shallower pretrained model which is RESNET18 of pytorch (because there isn't a pretrained RESNET18 in Keras applications) where i got a more precise model on test set which is the most important (93% train accuracy VS 88% test accuracy).
After training & testing the model, i added opencv library to the program to split the mommogram image into sub images (shape=(299*299) each), than evaluating them (with the trained model) and finally adding a red square to the sub image that have the higher probability between those who raised True "There is tumor".
PS: I programed this application to detect only one tumor in case thers is more than one, and the purpose behind that is to avoid negative false, so if there is no tumor there is no red square and if there is tumors there is one red sqare for the tumor with higher probability.
